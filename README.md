
# Copy Roblox Path

A simple extension that includes a nifty capability to copy the directory of any file by simply right clicking it, and pressing "Copy Roblox Path".

It's that simple!

## Get Started

- Install the extension through the marketplace at [this link](https://marketplace.visualstudio.com/items?itemName=waterbottlebag.copyrobloxpath) or Install the extension through our release file

## How to Use

- Right click any Lua or Luau file, then click "Copy Roblox Path".

![yes](https://github.com/user-attachments/assets/86e2ce55-f9e6-4d3d-952f-0a6bbaa01808)

- From there, it will be copied to your clipboard, and you can paste it wherever you want!
<img width="800" height="646" alt="image" src="https://github.com/user-attachments/assets/ae829e89-3c33-4eba-8a67-6db31176f08a" />

- Voila, you've instantly copied your path straight from the explorer, something impossible in Roblox Studio.
